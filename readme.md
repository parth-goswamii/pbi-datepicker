# Summary Date Range Filter

This custom visual is a UI prototype for date range selection.

- Single Current Period panel.
- Quick range presets (Today, Last 7 days, etc.).
- Calendar grid click-to-select range.
- Manual From / To date inputs.
- Day count summary.
- Apply button writes the chosen period(s) into the visual's persisted properties so you can read them in the Format pane.

Build steps:
1. `npm install`
2. `npx pbiviz package`
The packaged .pbiviz will appear under `.tmp\drop\`.
